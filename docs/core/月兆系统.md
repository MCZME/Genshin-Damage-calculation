# 月兆系统 (Lunar Omen System)

## 概述

月兆系统是一个队伍级别的增益系统，与月曜反应系统配合使用。

**专有名词**：
- 月兆 Moonsign
- 月兆·初辉 Moonsign: Nascent Gleam
- 月兆·满辉 Moonsign: Ascendant Gleam

## 月兆角色判定

月兆角色都有对应的天赋说明（通常在固有天赋中标注），可通过 `has_moonsign_talent` 属性判定。

---

## 月兆等级

| 等级 | 名称 | 触发条件 |
|------|------|----------|
| 0 | 无 | 队伍中无月兆角色 |
| 1 | 月兆·初辉 | 队伍编入 1 名月兆角色 |
| 2 | 月兆·满辉 | 队伍编入 2 名月兆角色 |

---

## 等级效果

### 月兆·初辉

队伍中的所有角色获得「月兆·初辉」效果标记。

具体增益由各角色技能/天赋自行实现。

### 月兆·满辉

队伍中的所有角色获得「月兆·满辉」效果标记，月兆角色的部分技能与天赋效果获得强化。

**注意**：除非另行说明，否则月兆等级为满辉时，也会触发「月兆·初辉」的对应效果。

---

## 非月兆角色增益

非月兆角色施放元素战技或元素爆发后的 20 秒内，基于角色自己的元素类型，使附近所有角色造成的月曜反应伤害提升至多 36%。

### 叠加规则

该效果**无法叠加**。当队伍中存在多位非月兆角色时，后一位的加成效果会**覆盖**前一位。

### 计算公式

| 元素类型 | 计算方式 |
|----------|----------|
| 火元素、雷元素、冰元素 | 每 100 点攻击力 → +0.9% |
| 水元素 | 每 1000 点生命值上限 → +0.6% |
| 岩元素 | 每 100 点防御力 → +1% |
| 风元素、草元素 | 每 100 点元素精通 → +2.25% |

**上限**：36%

---

## 实现要点

1. **月兆等级判定**：队伍组建时统计月兆角色数量
2. **效果标记**：为队伍中所有角色添加 `moonsign_nascent` 或 `moonsign_ascendant` 状态
3. **非月兆增益**：
   - 监听 `AFTER_SKILL` 和 `AFTER_BURST` 事件
   - 检查角色是否为非月兆角色
   - 计算并覆盖之前的增益值
4. **角色实现**：各月兆角色在自己的技能/天赋中检查 `moonsign_nascent` 或 `moonsign_ascendant` 状态来实现具体增益
