# 月曜反应 (Lunar Reactions)

## 概述

月曜反应是元素反应的**第三种类别**，与增幅反应、剧变反应并列。

**核心特点**：在特定角色存在时，将原有反应转化为月曜变体，产生独特的月曜伤害。

## 触发条件

月曜反应需满足以下条件：

| 条件 | 描述 |
|------|------|
| 队伍配置 | 队伍中有特定角色 |
| 触发来源 | 仅角色触发的反应可转化为月曜反应 |
| 敌人/环境 | 触发的反应仍为原反应 |

**转化规则**：角色触发的对应反应 → 替换为月曜反应

---

## 月曜伤害特性

月曜伤害是一种独特的伤害类型：

| 特性 | 月曜伤害 | 直接伤害 | 剧变伤害 | 真实伤害 |
|------|:--------:|:--------:|:--------:|:--------:|
| 可暴击 | ✅ | ✅ | ❌ | ❌ |
| 享受增伤加成 | ❌ | ✅ | ❌ | ❌ |
| 无视防御 | ✅ | ❌ | ❌ | ✅ |

**总结**：月曜伤害 = 可暴击 + 不享受增伤 + 无视防御 + 无附着

## 伤害计算公式

### 通用公式

```
月曜伤害 = 基础伤害区 × 暴击区 × 抗性区 × 擢升区
```

### 基础伤害区

```
月曜基础伤害区 = 核心基础伤害 × 反应提升 + 附加伤害

核心基础伤害 = 属性 × 属性倍率 × 反应倍率 × (1 + 基础伤害提升/100%)

反应提升 = 1 + 月曜精通系数 + (月曜反应造成的伤害提升/100%)
```

### 精通系数对比

| 类型 | 公式 |
|------|------|
| 剧变精通系数 | `16 × 精通 / (精通 + 2000)` |
| 月曜精通系数 | `6 × 精通 / (精通 + 2000)` |

### 各反应倍率

| 反应类型 | 角色触发 | 反应触发（实体造成） |
|----------|:--------:|:--------------------:|
| 月绽放 | 1.0 | — |
| 月感电 | 3.0 | 1.8 |
| 月结晶 | 1.6 | 0.96 |

### 暴击区

```
暴击区(触发暴击) = 1 + 暴击伤害/100%
暴击区(期望值) = 1 + (暴击率 × 暴击伤害)/(100% × 100%)
```

### 抗性区

| 目标抗性范围 | 公式 |
|--------------|------|
| 0% ~ 75% | `1 - 元素抗性/100%` |
| < 0% | `1 - 元素抗性/(2 × 100%)` |
| > 75% | `1 ÷ (1 + 4 × 元素抗性/100%)` |

### 擢升区

```
擢升区 = 1 + 月曜反应伤害擢升/100%
```

> **说明**：擢升是非常稀有的独立增伤，获取方式极少（涉及五星命座），防御区理论上存在但无限趋近于 1。

### 等级基数

| 等级 | 基数 |
|------|------|
| 80 | 1077.4 |
| 90 | 1446.9 |
| 95 | 1561.5 |
| 100 | 1674.8 |

## 月绽放

> **一句话总结**：绽放反应的变体，额外生成「草露」资源，消耗草露可造成月曜伤害。

### 触发条件

**元素组合**：水 + 草

**队伍要求**：至少有以下角色中的 1 位：
- 奈芙尔
- 菈乌玛
- 哥伦比娅

### 与绽放反应的关系

月绽放**不改变**原有绽放机制，而是在其基础上**额外赋予**「草露」资源。

**绽放反应特点回顾**：

| 特点 | 描述 |
|------|------|
| 不共存反应 | 反应触发后，至少有一方被消耗殆尽 |
| 草克制水 | 1U 草可消耗 2U 水（消耗比 1:2） |
| 草原核生成 | 每次反应生成 1 枚草原核 |
| 草原核迸发 | 超过存在时间或数量限制时迸发，造成草元素范围伤害 |
| 二级反应 | 草原核受雷/火攻击触发超绽放/烈绽放 |
| 友伤减免 | 对角色自身伤害为正常值的 0.02 倍 |

### 草露机制

| 属性 | 描述 |
|------|------|
| 触发 | 月绽放反应时生成 |
| 上限 | 3 枚 |
| 持续恢复 | 触发后 2.5 秒内，每 2.5 秒提供 1 枚草露 |
| 刷新 | 再次触发月绽放时刷新持续时间 |

### 草露消耗

- 部分角色技能可消耗草露（整数枚，至少 1 枚）
- 消耗后，对应攻击造成**月绽放伤害**
- 部分角色技能可不消耗草露，直接造成月绽放伤害（如哥伦比娅战技）

### 丰穰之核

月绽放生成的特殊草原核变体：

| 属性 | 草原核 | 丰穰之核 |
|------|--------|----------|
| 存在时间 | 正常 | 仅 0.4 秒 |
| 数量上限 | 共享 | 共享 |
| 迸发范围 | 正常 | 6.5 米 |
| 反应倍率 | 不变 | 不变 |
| 伤害类型 | 剧变伤害 | 剧变伤害 |

### 月绽放伤害

**触发条件**：仅由特定角色技能造成（草原核伤害不算月绽放伤害）

**伤害特性**：可暴击、不享受增伤、无视防御、无附着

### 触发与伤害的关系

| 情况 | 结果 |
|------|------|
| 仅触发月绽放反应（无相关角色技能） | 仅草原核剧变伤害，无月绽放伤害 |
| 特定角色消耗草露/使用技能 | 造成月绽放伤害（月曜伤害） |

## 月感电

> **一句话总结**：感电反应的变体，生成「雷暴云」实体，攻击基于多角色附着的加权求和。

### 触发条件

**元素组合**：水 + 雷

**队伍要求**：至少有以下角色中的 1 位：
- 菲林斯
- 伊涅芙
- 哥伦比娅

### 感电反应机制回顾

| 属性 | 描述 |
|------|------|
| 攻击间隔 | 每 1 秒结算 1 次 |
| 触发者 | 取最新上水/雷附着的目标 |
| 传导攻击 | 对附近 13 米内（高度 3 米）承受水附着的目标 |
| 伤害冷却 | 同一触发者与受击方，每 0.5 秒至多 1 次 |
| 反应倍率 | 2.0 |

### 感电 vs 月感电 对比

| 属性 | 感电 | 月感电 |
|------|------|--------|
| 攻击间隔 | 1 秒 | 2 秒 |
| 索敌范围 | 13米×3米 | 8米×8米 |
| 触发者 | 最新附着者 | 所有附着贡献者加权 |
| 特殊附着 | 电元素附着 | 无（雷暴云实体） |
| 伤害类型 | 剧变伤害 | 月曜伤害 |
| 反应倍率 | 2.0 | 1.8(反应)/3.0(角色) |

### 月感电触发与消耗机制

**触发条件**：

| 条件 | 感电 | 月感电 |
|------|------|--------|
| 触发方式 | 仅攻击行为触发 | 仅攻击行为触发 |
| 反应类型 | 共存反应 | 共存反应 |
| 附着衰减 | 保留原衰减速率 | 保留原衰减速率 |

**非触发情况**：
- 角色自身附着（如多莉除外）→ 保留雷水共存，不触发月感电
- 扩散同时附着雷水（无确定后手元素）→ 不触发月感电

**消耗机制**：
- 每次感电/月感电攻击：同时消耗目标身上各 0.4U 雷与水附着
- 该行为不视为触发反应

### 雷暴云机制

**生成**：触发月感电时，取代原先的特殊雷附着（电元素），生成**雷暴云**实体。

**属性**：

| 属性 | 描述 |
|------|------|
| 索敌范围 | 半径 8 米，高度 8 米 |
| 攻击间隔 | 每 2 秒结算 1 次 |
| 攻击时点 | 第 0.25 秒、2.25 秒、4.25 秒... |

**攻击条件**：
- 目标在索敌范围内
- 目标承受水雷附着
- 至少 1 种附着来自角色

**移动机制**：
- 雷暴云尝试向角色位置移动
- 与生成位置距离不超过 5 米

**存在条件**：
- 目标在雷暴云索敌范围内
- 目标身上有来自角色的附着

**数量限制**：
- 两朵雷暴云距离过近时，其中一朵会被摧毁

### 月感电伤害类型

| 类型 | 来源 | 描述 |
|------|------|------|
| 角色月感电 | 特定角色技能 | 视为月感电反应伤害的雷元素伤害 |
| 反应月感电 | 雷暴云攻击 | 由雷暴云造成的伤害 |

### 角色月感电伤害计算

**反应倍率**：3.0

```
核心基础伤害 = 属性 × 属性倍率 × 3.0 × (1 + 基础伤害提升/100%)
```

**属性与倍率**（因角色而异）：

| 角色 | 技能 | 属性 | 倍率来源 |
|------|------|------|----------|
| 菲林斯 | 雷霆交响 | 攻击力 | 攻击倍率 |
| 哥伦比娅 | 引力干涉·月感电 | 生命值上限 | 生命倍率 |

### 反应月感电（雷暴云攻击）伤害计算

**核心差异**：
- 基础伤害的「属性×属性倍率」部分 → 换为**等级基数**
- 所有记录的元素附着对应角色都参与计算
- 最终伤害取各伤害组分的**加权总和**
- 反应倍率：**1.8**

**单组分基础伤害区**：

```
月感电基础伤害区 = 等级基数 × 1.8 × (1 + 基础伤害提升/100%) × 反应提升 + 附加伤害
```

**加权求和（最终伤害）**：

所有被记录元素附着的角色分别计算「伤害组合」，按从大到小排序后：

```
最终伤害 = 最高伤害组合 + (第二高伤害组合 ÷ 2) + (其余伤害组合之和 ÷ 12)
```

### 附着来源追踪机制

- 即使角色造成的同元素附着会按叠加规则覆盖，仍分别计算各角色造成的附着剩余量
- 同一角色贡献多种元素（如风元素角色贡献水+雷）时，仍只算 1 个伤害组分

**示例**：

雷暴云攻击时，目标承受：
- 角色A：0.2 雷
- 角色B：0.7 水
- 角色C：0.8 雷 + 0.3 水

叠加后表现为：0.8 雷 + 0.7 水共存附着

参与伤害组分计算的角色：A、B、C（共3个组分）

### 暴击计算说明

- 各组分独立判定暴击
- 涉及 ≥2 个组分时，无法用简单期望公式计算
- 需先算出各组分暴击/未暴击结果，再按加权求和规则与暴击率计算期望

### 哥伦比娅大招特殊效果

处于哥伦比娅大招「月之领域」中的角色触发月感电时：

| 效果 | 描述 |
|------|------|
| 额外雷击概率 | 33% |
| 额外雷击特性 | 不额外消耗元素 |

### 月感电伤害来源

| 角色 | 技能/来源 |
|------|-----------|
| 菲林斯 | 特殊大招「雷霆交响」（月兆·满辉 + 附近有雷暴云强化） |
| 伊涅芙 | 突破天赋1额外攻击（攻击时附近有雷暴云触发） |
| 哥伦比娅 | 战技攒够引力值后触发「引力干涉」 |

**命座额外伤害**：伊涅芙 2命、6命；菲林斯 2命

## 月结晶

> **一句话总结**：结晶反应的变体，生成「月笼」攻击实体，每 3 次触发进行谐奏攻击。

### 触发条件

**元素组合**：岩 + 水

**队伍要求**：至少有以下角色中的 1 位：
- 兹白
- 哥伦比娅

**触发规则**：
- 角色触发的水元素结晶反应 → 替换为月结晶反应
- 火/雷/冰元素的结晶反应 → **保持不变**
- 敌人/环境触发的对应反应 → 仍为水结晶

### 结晶反应机制回顾

| 属性 | 描述 |
|------|------|
| 触发方式 | 仅后手为岩元素的攻击行为可触发 |
| 反应类型 | 不共存反应（至少一方被消耗殆尽） |
| 元素克制 | 岩被火/水/雷/冰克制（消耗比 2:1） |
| 触发冷却 | 火/水/雷/冰结晶：同一目标每 1 秒至多触发 1 次 |

### 结晶 vs 月结晶 对比

| 属性 | 结晶 | 月结晶 |
|------|------|--------|
| 触发冷却 | 1 秒 | **无** |
| 克制关系 | 岩被克制（2:1） | 相同 |
| 产物 | 晶片（护盾） | 月笼（攻击实体） |
| 数量上限 | 3 枚 | 3 枚 |
| 存在时间 | 17.5 秒 | 9 秒（无攻击则销毁） |
| 伤害类型 | 无 | 月曜伤害 |
| 索敌范围 | 无 | 半径 12 米，高度 5 米 |

### 月笼机制

**产物**：月笼（攻击性实体，非防御性）

#### 生成规则

| 目标在月笼索敌范围内 | 生成数量 |
|----------------------|----------|
| 0 枚月笼范围内 | 3 枚 |
| 1 枚月笼范围内 | 2 枚 |
| 2 枚月笼范围内 | 1 枚 |
| 3 枚月笼范围内 | 0 枚 |

**生成位置**：
- 生成 2~3 枚月笼时：在目标周围 3~3.5 米半径圆周上等距排布
- 若某生成位置旁已存在其他岩造物：位置改变（更接近目标且不规则）

#### 月笼属性

| 属性 | 描述 |
|------|------|
| 类型 | 岩元素创造物 |
| 可移动 | 否 |
| 碰撞模型 | 无 |
| 岩造物公共上限 | **不参与** |

**不参与公共上限的岩造物**：旅行者·岩的岩嶂、卡齐娜的冲天转转、月笼

**参与公共上限的岩造物**：荒星、岩脊、阳华、璇玑屏、阿丑（至多 3 个）

#### 谐奏攻击机制

**触发条件**：每触发 3 次月结晶反应

**攻击方式**：
- 受角色攻击的某目标附近的**随机 3 枚月笼**各发射投射物
- 造成月曜伤害（反应月结晶）

**特殊说明**：
- 即使存在多枚月笼，也只有 3 枚会攻击
- 被攻击目标不一定是触发月结晶的目标
- 弹道可能被其他岩造物/目标阻挡

**初始限制**：生成月笼的那一次反应无法触发谐奏

#### 存活机制

| 条件 | 结果 |
|------|------|
| 9 秒内未进行谐奏攻击 | 月笼被销毁 |
| 进行谐奏攻击 | 重置存活计时 |

**演化趋势**：多目标场景下，中心位置月笼存活率高，边缘月笼易超时销毁 → 月笼向战场中心聚集

#### 溢出计数机制

| 属性 | 描述 |
|------|------|
| 溢出计数储存 | 最多储存 4 层 |
| 初始生成 | 不计入谐奏计数 |
| 后续触发 | 可能呈现 1~2 次反应即触发谐奏（消耗溢出计数） |

### 月结晶伤害类型

| 类型 | 来源 | 描述 |
|------|------|------|
| 角色月结晶 | 特定角色技能 | 视为月结晶反应伤害的岩元素伤害 |
| 反应月结晶 | 月笼谐奏攻击 | 由月笼造成的伤害 |

### 角色月结晶伤害计算

**反应倍率**：1.6

```
核心基础伤害 = 属性 × 属性倍率 × 1.6 × (1 + 基础伤害提升/100%)
```

### 反应月结晶（月笼谐奏）伤害计算

**核心机制**：
- 基础伤害的「属性×属性倍率」部分 → 换为**等级基数**
- 所有记录的元素附着对应角色都参与计算
- 最终伤害取各伤害组分的**加权总和**
- 反应倍率：**0.96**

**单组分基础伤害区**：

```
月结晶基础伤害区 = 等级基数 × 0.96 × (1 + 基础伤害提升/100%) × 反应提升 + 附加伤害
```

**加权求和（最终伤害）**：

```
最终伤害 = 最高伤害组合 + (第二高伤害组合 ÷ 2) + (其余伤害组合之和 ÷ 12)
```

### 附着参与规则

**参与条件**：只要角色参与了月笼谐奏所需的 3 次月结晶反应中的任意一次附着记录，就会参与**全部 3 次谐奏攻击**的伤害组分计算。

**示例**：

月笼谐奏攻击前，3 次月结晶反应：
- 第 1 次：角色A（水）+ 角色B（岩）
- 第 2 次：角色A（水）+ 角色C（岩）
- 第 3 次：角色A（水）+ 角色C（岩）

结果：3 次谐奏攻击的伤害组分均包含角色 A、B、C

### 暴击独立判定

- 各谐奏攻击的各伤害组分独立判定暴击
- 3 次谐奏攻击的最终伤害可能各不相同

**示例**（A、B 两角色参与）：
- 伤害1：A 暴击，B 不暴击
- 伤害2：A 不暴击，B 暴击
- 伤害3：A、B 均暴击

### 攻击范围

| 属性 | 描述 |
|------|------|
| 伤害范围 | 半径约 0.5 米 |
| 目标特性 | 几乎只能对单 |
| 多目标优化 | 范围攻击同时触发月结晶可提高谐奏频率 |

### 哥伦比娅大招特殊效果

处于哥伦比娅大招「月之领域」中的角色触发月结晶反应时：

## 复合反应机制

当队伍/实战环境中存在多元素时，需考虑复合反应问题：

| 类型 | 描述 |
|------|------|
| 经典复合反应 | 向共存附着施加单一后手元素 |
| 同时附着 | 向目标同时施加多个后手元素（多目标扩散） |

由于月绽放不改变核心机制，本节只介绍月结晶及月感电相关的复合反应。

### 月结晶的复合反应

#### 触发冷却差异

| 反应 | 触发冷却 |
|------|----------|
| 火/水/雷/冰结晶 | 1 秒（共用） |
| 月结晶 | 无冷却 |

#### 场景一：雷水共存 + 岩

**不满足月结晶触发条件**：
- 即使岩元素过量，也只能触发雷结晶
- 不会继续触发水结晶

**满足月结晶触发条件**：
- 过量岩元素可直接消耗完雷再继续消耗水
- 同时触发：雷结晶 + 月结晶
- 生成：雷晶片 + 月笼

**岩附着频率高的策略**：
- 引入后台雷角色
- 处理永久自附着雷目标（如雷音权现，4元素量永久雷附着）

**示例**：雷过量时，1 秒内连续施加两次岩附着
- 第一次：触发雷结晶（进入 1 秒冷却）
- 第二次：雷结晶冷却中，只能触发月结晶

#### 场景二：冻水共存 + 岩

**冻水共存实现方式**：
- 为具有冻元素附着（非冻结状态）的目标附着水
- 或冻标记藏水

**满足月结晶触发条件**：

施加过量岩元素：
- 先触发月结晶
- 后触发冰结晶

**不满足月结晶触发条件**：
- 只会发生水结晶
- 不会继续发生冰结晶
- 若为冻结藏水：碎冰 + 水结晶

**策略优势**：月结晶相关队伍几乎不怕永久自附着冻的目标，且强岩可额外获取冰元素结晶盾。

### 月感电的复合反应

月感电相对感电在复合反应方面的差别没有月结晶显著，但共存反应本身的复杂性与同时附着可能性使得实际情况非常复杂。

#### 经典复合反应

##### 场景一：雷水共存 + 后手元素

无论是否满足月感电触发条件，施加后手元素发生的反应（月结晶除外）与原规则完全一致。

**实战意义**：火C蒸发队伍与物伤（超导）相关队伍

**消耗频率差异**：
- 月感电消耗频率比感电更低
- 容易使后手火/冰元素消耗更多雷以降低挂水需求

**示例**：

队伍配置：
- 持续、高频弱雷附着（折算损耗后最多 0.8U）
- 相对低频的水附着
- 施加弱火（1U）

结果：
- 同时触发超载和蒸发
- 最少仅消耗 0.1U 水

**策略**：雷能为蒸发节约的水元素量往往多于月感电本身对水的消耗，可在保证挂火位蒸发前提下带来月感电伤害。

**注意**：挂火过多会影响月感电出伤效率。

##### 场景二：冻结藏水 + 雷

**规则**：冻结状态下或存在冻标记的目标禁止发生感电/蒸发反应，对月感电同样生效。

**结果**：

向冻结藏水目标施加过量雷元素：
- 若后手元素所属攻击不为钝击：只能发生超导
- 若为钝击：先碎冰，再进行后续反应
- 不会发生感电/月感电

##### 场景三：激雷共存 + 水

**施加过量水元素**：
- 先触发绽放
- 后触发感电
- 剩余水因后手不残留规则被清除
- 反应均来不及消耗雷元素

**满足月感电触发条件**：
- 月感电延时约 0.25~0.3 秒后一起攻击与消耗
- 只生成雷暴云，不造成伤害

**不满足月感电触发条件**：
- 感电触发时即进行攻击，之后才消耗
- 造成一次感电攻击

#### 同时附着

##### 场景一：无附着 + 同时附着水+雷

**结果**：无论是否满足月感电触发条件，均不发生反应

**原因**：
- 感电/月感电触发需后手附着为攻击行为
- 同时附着不分先后手
- 直接导致水雷共存附着

##### 场景二：过量水 + 同时附着冰+雷

**同时附着处理机制**：

1. 抽取一个元素参与反应
2. 处理反应瞬间机制（含不共存反应消耗、造成感电攻击/生成雷暴云，不含造成特殊附着）
3. 若该元素发生消耗则按后手不残留规则清除，否则折算损耗后进入附着列表
4. 抽取另一个元素处理相同流程
5. 结算附着列表中的元素（叠加或反应）
6. 造成先前步骤未正常发生的反应效果（特殊附着、若满足条件则进行感电/月感电消耗）

**后手元素抽取顺序**：
- 目标以自身面朝方向的 12 点钟方向起逆时针检测攻击来源
- 按顺序抽取对应后手元素
- 夹角相同时，先抽取离目标最近的攻击来源对应后手元素

**情况A：先抽取冰元素**

处理流程：
1. 冰与水发生冻结反应，按后手不残留规则清除
2. 抽取雷元素与剩余水发生感电/月感电反应
3. 雷元素未造成消耗，折算损耗后施加

**满足月感电触发条件**：
- 只生成雷暴云
- 在剩余水和已附着雷共存基础上施加冻元素
- 发生超导反应
- 若冻不少于雷则无法残留雷附着，无法产生月感电攻击及消耗
- 多余冻元素不残留

**不满足月感电触发条件**：
- 造成一次感电攻击
- 将电元素加入附着列表
- 在水雷共存基础上先附着冻元素后附着电元素
- 冻元素因未能消耗完雷或后手不残留而清除
- 最终残留电元素（冻不足时同时残留少量雷）以继续发生感电消耗

**情况B：先抽取雷元素**

处理流程：
1. 先发生感电/月感电触发判定
2. 后进行冰元素对先手水元素的冻结消耗

**满足月感电触发条件**：最终结果与先抽取冰元素相同

**不满足月感电触发条件**：
- 在水雷共存基础上先附着电元素后附着冻元素
- 冻元素过量时可使最终仅残留水附着，无法继续发生感电消耗

##### 场景三：少量水 + 同时附着冰+雷（冰多于水）

**先抽取冰元素**：
- 无法发生感电/月感电
- 最终视雷/冻元素量决定是否残留雷

**先抽取雷元素**：
- 先发生感电/月感电触发判定
- 后使水元素被冰消耗完
- 相比先抽取冰，多出一次感电攻击或生成雷暴云
