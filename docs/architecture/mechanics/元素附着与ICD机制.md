# 架构设计：元素附着与ICD机制

## 1. 元素附着参数
- **元素量 (Gauge)**：1U/2U/4U 离散等级及其衰减斜率。
- **附着消耗**：反应时的 0.8x 消耗规则。

## 2. ICD (内部冷却时间) 控制
利用“衰减序列”与“衰减标签”实现：
- **标签匹配**：同一标签下的攻击共享 ICD 计数器。
- **计数器机制**：通常为“3次打击”或“2.5秒”重置附着。
- **独立附着**：特殊技能（如香菱大招）通过独立标签实现每一段都能附着。

## 3. 元素感应协议 (IReactable)
- **实体接口**：所有 `CombatEntity` 必须实现对元素碰撞的响应。
- **环境交互**：非怪物实体（如火把、水面、草原核）通过此协议参与元素反应循环。
