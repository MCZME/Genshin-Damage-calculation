# UI美术设计规范 (V2.2 - MD3 Panes 融合版)

本规范定义了“原神伤害计算器”工作台的视觉语言，旨在通过 **Material Design 3 (MD3)** 的面板化 (Panes) 架构与 **《原神》 (Genshin Impact)** 的沉浸式游戏美学相结合，打造一个既专业又具代入感的生产力工具。

---

## 1. 设计哲学 (Design Philosophy)

*   **理性的骨架 (MD3 Panes)**：利用独立面板 (Independent Panes) 区分功能区，强调层级感和模块化。
*   **感性的皮肤 (Genshin)**：引入磨砂玻璃材质 (Backdrop Blur)、元素艺术色点缀、以及非对称装饰细节。

---

## 2. 色彩系统 (Color System)

### 2.1 全局基础色 (Global Tokens)
*   **底座 (Surface)**：`#0B0E18` (极深蓝黑，星空底色)。
*   **面板背景 (Pane Surface)**：`rgba(15, 23, 42, 0.6)`，配合 `backdrop-blur-xl`。
*   **文本 (On-Surface)**：
    *   主要：`#ECE5D8` (暖白，原神文字标准色)。
    *   次要：`#94A3B8` (灰蓝)。

### 2.2 动态元素主题 (Dynamic Element Theming)
UI 的 **Primary (主色)** 随当前角色的元素属性动态切换：火红 (Pyro)、水蓝 (Hydro)、草绿 (Dendro) 等。

---

## 3. 布局架构：Panes (面板化)

工作台不再采用平面线条分割，而是由多个“悬浮”面板组成。

### 3.1 面板规范 (Pane Specs)
*   **圆角 (Radius)**：所有主要面板采用 **28px** 的超大圆角。
*   **间距 (Gap)**：面板之间保持 **16px (1rem)** 的固定间距，透出底部的星空色。
*   **边框 (Border)**：`1px solid rgba(255, 255, 255, 0.08)`，模拟磨砂玻璃边缘。
*   **阴影 (Shadow)**：`0 8px 32px 0 rgba(0, 0, 0, 0.3)`，增强悬浮感。

### 3.2 结构锚点
*   **顶栏 (Header)**：全宽 (Full-width) 玻璃基座，作为页面稳定的视觉锚点。
*   **底栏 (Footer)**：厚重的独立悬浮控制岛，提供核心仿真操作及系统状态显示。

---

## 4. 组件规范 (Component Standards)

### 4.1 玻璃卡片 (Genshin Card)
*   **圆角**：24px。
*   **交互**：Hover 时边框变亮并向主色调偏移，伴随轻微的位移反馈。

### 4.2 控制按钮 (Prominent FAB)
*   **仿真按钮**：大尺寸胶囊形状，使用主色调填充，带有强烈的投影 (Shadow Glow)。

### 4.3 输入控件 (Input Fields)
*   **风格**：MD3 Outlined 风格，极细边框，聚焦时边框色变为动态元素色。

---

## 5. 交互动效 (Motion)

*   **材质反馈**：切换角色时，所有面板的主色调通过 400ms 的平滑过渡同步变换。
*   **战术雷达**：右栏面板内置旋转扫描线动效，模拟战术侦察感。
