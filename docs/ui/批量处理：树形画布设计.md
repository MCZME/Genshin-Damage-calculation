# UI 设计：批处理编辑器 (V3.4)

## 1. 设计核心

采用 **"可视化树形画布 + 右侧检视面板"** 的双区域布局。通过 `MindMapCanvas` 构建变异树，`NodeInspectorPanel` 编辑节点规则，实现直观的批处理配置体验。

## 3. 树形画布设计

### 3.1 画布数据模型 (`MindMapCanvasData`)

```python
@dataclass
class MindMapCanvasData:
    root: BatchNode                    # 树根节点
    selected_node_id: str              # 当前选中节点ID
    node_index: dict[str, NodeViewModel]  # 节点ID到视图模型的映射
    drawer_open: bool                  # 添加抽屉是否打开
    drawer_parent_id: str              # 抽屉父节点ID
    drawer_anchor_x: float             # 抽屉锚点X坐标
    drawer_anchor_y: float             # 抽屉锚点Y坐标

    # 布局参数
    x_gap: int = 300                   # 水平间距
    y_gap: int = 170                   # 垂直间距
    base_x: int = 320                  # 起始X坐标
    base_y: int = 140                  # 起始Y坐标
```

### 3.2 布局算法

采用 **自底向上测量 + 自顶向下放置** 的布局策略：

```python
def measure(node: BatchNode) -> int:
    """计算节点跨度（子树覆盖的叶子数）。"""
    if not node.children:
        return 1  # 叶子节点跨度为1
    return max(1, sum(measure(child) for child in node.children))

def place(node: BatchNode, depth: int, leaf_offset: int) -> None:
    """根据跨度计算节点居中位置。"""
    span = spans[node.id]
    center_index = leaf_offset + (span - 1) / 2  # 居中计算
    positions[node.id] = (
        base_x + depth * x_gap,
        base_y + center_index * y_gap,
    )

    next_offset = leaf_offset
    for child in node.children:
        child_span = spans[child.id]
        place(child, depth + 1, next_offset)
        next_offset += child_span
```

**布局效果**：
- 根节点位于左侧，子节点向右延伸
- 每层节点按跨度居中排列
- 兄弟节点间自动保持均匀间距

### 3.3 连线绘制

使用三次贝塞尔曲线 (Cubic Bezier) 绘制父子节点间的连接线：

```python
cv.Path.CubicTo(
    x + 270, y + card_mid_y,           # 控制点1（父节点右侧）
    child_x - 60, child_y + card_mid_y, # 控制点2（子节点左侧）
    child_x, child_y + card_mid_y,      # 终点
)
```

**视觉参数**：
- 颜色：`#D1A2FF` (明紫色) ，透明度 72%
- 线宽：`2.6px`
- 样式：平滑曲线，无箭头

## 4. 节点卡片设计 (`NodeCard`)

### 7.3 视图模型同步机制

`_sync_view_models()` 方法遍历项目树，将 `BatchNode` 转换为视图模型：

```python
def _sync_view_models(self) -> None:
    node_vms: dict[str, NodeViewModel] = {}

    def walk(node: BatchNode) -> None:
        vm = NodeViewModel(
            id=node.id,
            name=node.name,
            kind=node.kind,
            target_path=self._format_path(node.rule.target_path),
            value_text=self._format_value(node.rule.value),
            ...
        )
        node_vms[node.id] = vm
        for child in node.children:
            walk(child)

    walk(self.project.root)
    self.node_vms = node_vms
    self.canvas_data = MindMapCanvasData(...)
    self.inspector_vm = self._build_inspector_vm(selected)
```

## 8. 交互规范

### 8.1 画布交互

- **缩放**: 支持 0.45x ~ 1.8x 缩放
- **平移**: 无边界限制自由平移
- **点击空白**: 取消选中，收起检视面板

### 8.2 节点操作约束

| 操作 | ROOT | RULE | RANGE_ANCHOR | 受控节点 |
|------|------|------|--------------|---------|
| 删除 | ❌ | ✅ | ✅ | ❌ |
| 添加子节点 | ✅ | ✅ | ✅ | ✅ |
| 编辑规则 | ❌ | ✅ | ❌ | ❌ |
| 编辑区间 | ❌ | ❌ | ✅ | ❌ |

### 8.3 视觉反馈

- **执行中**: 进度条 + 状态文本 + 禁用编辑
- **错误**: 红色错误提示文本
- **完成**: 显示统计摘要（平均DPS、最大DPS等）
