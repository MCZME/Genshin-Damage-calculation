# 视觉设计指南 (V2 - 沉浸式风格)

## 1. 动态元素配色 (Element Dynamics)
UI 的强调色应根据当前选定角色的元素属性自动切换。

| 元素 | 主色调 | 阴影光晕 (Glow) |
| :--- | :--- | :--- |
| 火 (Pyro) | `#FF4D4D` | 强度 0.8, 模糊 15 |
| 水 (Hydro) | `#3399FF` | 强度 0.8, 模糊 15 |
| 风 (Anemo) | `#66FFCC` | 强度 0.8, 模糊 15 |
| 雷 (Electro) | `#CC66FF` | 强度 0.8, 模糊 15 |
| 草 (Dendro) | `#99FF33` | 强度 0.8, 模糊 15 |
| 冰 (Cryo) | `#99FFFF` | 强度 0.8, 模糊 15 |
| 岩 (Geo) | `#FFCC33` | 强度 0.8, 模糊 15 |

## 2. 原子组件视觉规范

### 2.1 属性输入框 (StatInputField)
- **背景**: `WHITE` 不透明度 0.05。
- **圆角**: 6px。
- **交互**: 聚焦时产生元素色描边与外发光。
- **特性**: 支持根据属性类型（数值/百分比）动态展示后缀图标。

### 2.2 资产卡片 (AssetCard / Picker Card)
- **比例**: 1:1 正方形 (80x80) 或 0.72 纵向比例。
- **稀有度渐变**:
  - 5星 (金色): `#E9B053` 到 `#D3962F`。
  - 4星 (紫色): `#AF8FE6` 到 `#8A69C4`。
- **特征**: 中心显示图标，底层可叠加元素文字水印。

### 2.3 角色/目标槽位 (Slot)
- **选中态**: 侧边产生向右平移 `Offset(0.04, 0)` 的动态效果，伴随元素色辉光。
- **非选中态**: 采用低透明度背景，保持低视觉干扰。

### 2.4 圣遗物插槽 (ArtifactSlot)
- **背景**: `GenshinTheme.SURFACE_VARIANT` (深色卡片)。
- **布局**: 紧凑型五轴并列，宽度固定 `215px`。
- **配色**: 标题颜色同步成员元素主题色。

## 3. 材质与动画
- **核心配色**: 采用深色背景 `#1A1625` (Astral Deep Purple)。
- **玻璃拟态**: 对全局 Header/Footer 应用 20px 模糊。
- **动画曲线**: 统一采用 `ft.AnimationCurve.DECELERATE` 或 `EASE_OUT_BACK`，时长 300-400ms。
