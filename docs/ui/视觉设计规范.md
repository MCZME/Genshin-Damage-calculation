# 视觉设计规范 (V4.0 - Artifact Prime)

> 本文档定义了仿真引擎 V4.0 的核心视觉语言，旨在复刻《原神》高品质艺术质感，同时兼顾工程数据的可读性。

---

## 1. 核心艺术风格 (Visual Identity)

### 1.1 材质：嵌套式金属玻璃 (Artifact Prime)
V4.0 舍弃了单一的 Border 属性，采用“嵌套收缩 (Nested Inset)”算法来实现完美的平滑金色边框。
*   **外层容器**: 使用 `ft.LinearGradient` 模拟金属光泽（`GOLD_LIGHT` 到 `GOLD_DARK`）。
*   **内层核心**: 通过 `padding: 2` 实现 2px 的物理边框，核心背景采用纯净深紫色 (`#1A1625`)。
*   **圆角逻辑**: 外层 34px，内层 32px，确保边缘厚度在转角处绝对均匀。

### 1.2 光影与氛围 (Lighting & Mood)
*   **顶部光晕**: 所有核心容器（如磁贴、对话框）顶部必须包含一个 100-150px 高的金色渐变层 (`0.15` 不透明度)，模拟垂直向下的照明感。
*   **装饰暗纹**: 背景底层应叠加极低透明度 (`0.01-0.02`) 的星形或几何暗纹图标，增加视觉深度。
*   **磨砂模糊**: 全局推荐模糊参数为 `Blur(15, 15)`，透明度控制在 `0.2-0.4` 之间以保持通透感。

## 3. 组件视觉标准

### 3.1 分析磁贴 (TileContainer)
*   **边框**: 嵌套式金色高光。
*   **圆角**: 顶部主圆角 32px，其余边缘 6px（非对称美学）。
*   **交互**: 悬停时外层渐变亮度提升 150%，模拟金属反光。

### 3.2 顶部标尺 (GlobalScrubber)
*   **风格**: 实色沉底，不抢占视觉重心。
*   **背景**: `#1E1A2A` (Solid Dark)。
*   **细节**: 使用 `Consolas` 等宽字体显示时间，消除数字跳变带来的廉价感。

---

## 4. 动画规范 (Motion)
*   **过渡曲线**: 优先使用 `ft.AnimationCurve.DECELERATE` (300ms) 实现平滑的展开。
*   **强反馈**: 对于全屏切换或历史记录弹出，使用 `ft.AnimationCurve.EASE_OUT_QUINT` (400ms) 以获得更优雅的入场感。
*   **渐变动画**: 文字淡入淡出统一使用 `animate_opacity=300`。
