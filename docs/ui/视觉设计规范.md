# 视觉设计规范 (V4.5 - Pro Plus)

> 本文档定义了仿真引擎 V4.5 Pro 的核心视觉语言。强调数据的高对比度、物理发光质感与精密的工业复盘美学。

---

## 1. 物理层 (Physical Layer)

### 1.1 边框与抗锯齿 (1.5px Pro Border)
自 V4.5 起，废弃了复杂的嵌套 Inset 算法，转而采用更高性能且视觉更锐利的 **“单层增强边框”**：
*   **宽度**: `1.5px`。该宽度在 Flet/Flutter 引擎下能获得最优的像素覆盖，消除 1px 细边的频闪问题。
*   **抗锯齿补偿**: 每个磁贴必须具备一个 `BoxShadow(blur_radius=20, spread_radius=-10)`。该阴影在逻辑边缘下方产生微弱的半透明过渡带，能完美抹平直线与圆角衔接处的渲染毛刺。
*   **圆角 (Radius)**: 统一为 `14px`。

### 1.2 材质与光影 (The Glow Material)
*   **色彩呼应 (Color Echoing)**: 磁贴边框色必须由内容主题色 (theme_color) 动态生成。
*   **垂直渐变**: 背景采用双色垂直渐变。起始色 (`gradient_top`) 通常为主题色的极暗 Tint，终止色固定为基准背景色。
*   **脉冲呼吸**: 顶层身份区的指示灯 (Pulse Dot) 在 Hover 时会产生与主题色一致的辉光 (`BoxShadow`)，模拟系统活跃状态。

## 3. 组件视觉标准

### 3.1 三层空间模型 (Three-Layer Layout)
所有容器型组件必须遵循以下三段式结构：
1.  **Identity Bar (H=30px)**: 极简身份标识与悬浮功能集群。
2.  **Hero Area (Expand)**: 核心内容。数值采用 `w900` 极粗字重，图表剥离所有修饰性边框。
3.  **Meta Footer (H=20px)**: 辅助元数据。字体大小统一为 `8px-9px`。

### 3.2 交互集群 (Action Cluster)
*   **尺寸**: 每个动作按钮固定为 `24x24px`，图标 `12px`。
*   **显隐**: 默认 `opacity=0`，Hover 时通过 `animate_opacity=200` 平滑显现。
*   **颜色**: 默认 `WHITE38`，关闭按钮专用 `RED_400`。

---

## 4. 动画规范 (Motion)
*   **进入曲线**: `DECELERATE` (300ms) - 用于磁贴状态切换。
*   **位移曲线**: `EASE_OUT_EXPO` (600ms) - 用于网格磁贴重排。
*   **交互反馈**: 按钮 Hover 采用 `overlay_color` 反馈，而非强行变色。
