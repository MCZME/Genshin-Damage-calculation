# UI - 伤害分布可视化组件 (DamageDistributionVisualizer)

伤害分布可视化组件是战后分析工作台的核心视觉单元。它通过高精度的 Canvas 渲染和动态分层算法，将数万条仿真伤害记录转化为直观、可交互的战斗分布图。

## 1. 渲染架构：三阶动态缩放引擎

为了在同一画布中平衡“高频常规伤害”与“极少数爆发伤害”的展示细节，组件采用了 **三阶动态线性缩放 (3-Tier Scaling)** 技术：

### 1.1 分层逻辑
- **常规区 (Normal Zone)**: 占据画布底部 75% 的垂直空间。通过梯度断层算法自动识别并展示最核心的战斗伤害区间。
- **溢出区 (Burst Zone)**: 占据顶部 25% 的垂直空间。专门用于展示超过分割阈值（$T_{split}$）的峰值爆发伤害，避免由于极高数值导致常规伤害被压缩成直线。
- **过滤区 (Noise Filtering)**: 自动识别累积贡献度不足 5% 的低位数据，主图不予绘制以净化视觉。

### 1.2 分割算法 ($T_{split}$)
系统根据当前 session 的伤害分布特征自动计算分割点：
$$T_{split} = \min(v_{after}, \max(v_{before} \times 1.5, Max / 3.0))$$

## 2. 视觉规范 (Visual Identity)

组件遵循极致简约的设计原则，移除一切不必要的视觉噪音：

### 2.1 极简坐标系统 (Minimalist Axis)
- **有标注，无线条**: 移除所有横向网格线和参考线。
- **数值标签**: 在右侧内嵌显示关键数值标签：`PEAK`（全场最高）、`SPLIT`（分层阈值）、`MID`（区间中位）。
- **时间轴**: X 轴保持带刻度的微型时间轴，与底部的热力带紧凑排列。

### 2.2 背景分层设计
- **分层视觉**: 溢出区叠加一层全宽贴边的深海蓝 (`#1A1C2E`) 背景块，通过底部 `BorderSide` 形成自然的物理分割感。
- **柱体外观**: 所有伤害柱体统一采用 5px 填充宽度，依靠高度自然表达量级差异，不使用额外的描边或发光特效。

## 3. 交互与反馈

- **实时读数**: 左上角提供“帧伤害: [Value]”的实时动态读数。
- **Scrubber 联动**: 支持与全局时间轴同步。当用户拖动 Scrubber 时，分布图会自动高亮对应帧的数据列。
- **三维空间投影**: 点击分布图中的特定点，可联动开启 `CombatSpace` 的 3D 帧快照复盘。

## 4. 物理参数
- **绘图核心宽度**: 636px
- **面板总宽**: 656px
- **高度分配**: 240px (Canvas) / 6px (热力带) / 35px (时间轴)
