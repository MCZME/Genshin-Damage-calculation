# Gemini Project Context

This directory contains project-specific instructions, rules, and skills for the Gemini CLI agent.

## Structure
- `rules/`: Contains behavioral and architectural guidelines (`rules.md`).
- `skills/`: Placeholder for custom tool-like skills.
- `configs/`: Configuration files for the agent's tools.

## Usage
The files in this directory are used to provide context and guidance for the AI agent when working on this repository, ensuring consistency with the project's architecture and coding standards.
